Как общаться с Claude

Короткий человеческий гайд по работе с Claude Code в VS Code. Не учебник — просто чтобы знать что можно, а что нет, и как с ним разговаривать.

Главное правило

Говори как с человеком. Не нужно учить специальные команды или синтаксис. Пиши как друг другу пишешь в мессенджере: «слушай, можешь сделать то-то».

Claude поймёт. И сделает.

Можно на русском

Да. Полностью. Запросы, ответы, обсуждения — всё на русском.

Смешивать тоже можно. Например:

Создай файл index.html с базовой структурой

Или вперемешку:

Слушай, у меня тут какой-то error в консоли — посмотри в чём дело

Работает одинаково.

Можно кидать скрины

Если что-то не получается, проще показать чем объяснить. Сделай скриншот и:

  • Перетащи мышкой прямо в окно чата Claude Code

  • Или скопируй скрин (Cmd+Shift+Ctrl+4 на маке) и вставь (Cmd+V) в чат

Claude посмотрит картинку и скажет что видит. Можно показывать:

  • Ошибки из терминала или браузера

  • Дизайн который хочешь повторить

  • Скрин с настройками которые не понятно как заполнить

  • Любую картинку — он опишет что на ней

💡 Пример: Кидаешь скрин ошибки → пишешь: «что это значит?» → Claude объясняет и помогает починить.

Можно задать любой вопрос

Не только про код. Серьёзно, любой:

  • «Что такое Git?»

  • «Как работает интернет если совсем простыми словами?»

  • «Чем отличается HTML от CSS?»

  • «Почему мой сайт открывается медленно?»

  • «Как назвать кнопку чтобы было понятно?»

Claude — это не только инструмент для кода. Это эксперт по всему IT, к которому можно подойти с любым вопросом без стеснения.

Можно попросить создать файл

Один из самых полезных навыков. Просто скажи что нужно:

Создай файл about.html с моей биографией

Сделай папку images и положи туда пустой файл README

Напиши скрипт на Python который считает строки в файле

Claude создаст файл, напишет содержимое, покажет что получилось. Если что-то не нравится — скажешь, переделает.

Можно попросить изменить файл

Открыт файл, что-то нужно поменять — просто скажи:

Поменяй заголовок на "Привет"

Добавь в конец списка пункт про кошек

Сделай этот цвет красным а не синим

Claude найдёт нужное место и изменит. Не нужно объяснять где и как — он сам разберётся.

Можно попросить объяснить

Открыт чужой код или непонятный файл:

Объясни что делает этот файл

Что здесь происходит в строке 25?

Это безопасно или может что-то сломать?

Получишь объяснение простыми словами.

Шпаргалка: что можно говорить

Вот примеры команд на русском, которые Claude точно поймёт. Можешь скопировать и использовать:

Создание

  • «Создай новый проект для сайта»

  • «Сделай файл style.css со стилями для главной страницы»

  • «Добавь страницу контактов»

Изменения

  • «Поменяй цвет фона на бежевый»

  • «Сделай шрифт побольше»

  • «Перенеси этот блок наверх»

  • «Добавь анимацию при наведении на кнопку»

Сохранение и публикация

  • «Сохрани изменения в Git с сообщением "Обновил главную"»

  • «Запушь всё на GitHub»

  • «Опубликуй проект через GitHub Pages»

Понимание

  • «Объясни что делает этот код»

  • «Покажи структуру проекта»

  • «В чём разница между этими двумя файлами?»

Когда что-то сломалось

  • «Сайт не открывается, помоги разобраться»

  • «У меня ошибка [скрин], что делать?»

  • «Откати последние изменения, что-то сломалось»

Просто вопросы

  • «Что такое API?»

  • «Как мне выбрать шрифт?»

  • «Какие цвета хорошо смотрятся вместе?»

Чего Claude не умеет

Чтобы не разочаровываться:

  • Не помнит прошлые чаты. Каждый новый чат начинается с нуля. Если что-то важное — пиши заново или сохрани в файл CLAUDE.md.

  • Не лазит в интернет по умолчанию. Если нужны актуальные данные — попроси «погугли» или дай ссылку.

  • Не делает дизайн "вау" сам. Если хочешь красиво — покажи референс (скрин) и скажи «сделай похоже».

  • Может ошибаться. Особенно в мелочах. Всегда проверяй результат глазами, а не только верь на слово.

Главный лайфхак

Если не знаешь как сказать — просто опиши проблему словами.

Не нужно гуглить термины. Не нужно знать как это правильно называется. Скажи как умеешь:

У меня в браузере вот эта штука сверху съезжает вправо когда я уменьшаю окно. Помоги это починить.

Claude разберётся.

Если застрял

  1. Покажи скрин — почти всегда помогает

  2. Скажи "не получается" — Claude сам предложит варианты

  3. Попроси объяснить простыми словами — если ответ непонятный

  4. Попроси переделать — если результат не нравится

  5. Открой новый чат — если запутались окончательно

Главное

Claude Code — это не сложный инструмент с правилами. Это собеседник, который понимает что ты говоришь, помогает делать, и не устаёт от твоих вопросов.

Не бойся писать «глупые» вещи. Глупых вопросов нет — есть только не заданные.
