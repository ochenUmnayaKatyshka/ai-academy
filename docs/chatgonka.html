Как сделать свой
AI-чат в Telegram

↗

Простая инструкция с нуля. Делать почти ничего не придётся — Claude всё соберёт сам. Тебе нужно только дать ему задание и ключи.

1
Установи Claude
  1. Тебе нужен компьютер.
  2. Скачай Claude Code: claude.com/claude-code
  3. Открой на компьютере терминал:
    • Mac: нажми ⌘ + Пробел, напиши Terminal и нажми Enter.
    • Windows: нажми Win + R, напиши cmd и нажми Enter.
  4. Напиши команду: 
    claude
  5. Сразу вставь промпт из Шага 2 — Claude соберёт весь проект сам.
2
Вставь промпт

Скопируй промпт ниже и вставь его в Claude. Это точный промпт, по которому собирается этот бот.

Привет! Помоги мне сделать AI-чат в Telegram — точно такой:

Создай отдельную папку Gonkachat в моих Projects и собери в ней
универсального чат-бота на Python, который общается как ChatGPT.

Технические детали:
- Язык Python, библиотеки: python-telegram-bot, openai, python-dotenv.
  Важно: закрепи httpx==0.27.2 (иначе будет ошибка с 'proxies').
- Бот ходит к модели через Gonka Broker — это OpenAI-совместимый API.
  base_url: https://proxy.gonkabroker.com/v1
  модель MiniMax: MiniMaxAI/MiniMax-M2.7
- Все секреты (токен Telegram, ключ Gonka, base_url, имя модели)
  вынеси в файл .env.
- Бот помнит контекст последних сообщений диалога и имеет команды
  /start и /reset.
- Модель MiniMax возвращает свои размышления в тегах <think>...</think> —
  вырезай их из ответов, чтобы пользователю приходил только чистый текст.

После сборки помоги мне вставить мои ключи в .env, проверь связь
с Gonka и запусти бота для проверки.

Дальше Claude сам скажет, какие ключи ему нужны — переходи к Шагу 3.

3
Получи ключи Gonka
  1. Открой app.gonkabroker.com и залогинься через Google.
  2. Заведи ключ — нажми New key.
  3. Скопируй ключ ⚠️ он показывается только один раз!
  4. Скопируй URL.
  5. Скопируй имя модели — для этого проекта выбирай MiniMax.

Все эти данные нужно вставить в файл .env.

4
Создай бота в Telegram
  1. Открой в Telegram @BotFather.
  2. Нажми /newbot и придумай имя (должно заканчиваться на _bot).
  3. Скопируй токен и вставь его в файл .env.

🙂 Если запутался — попроси Claude: «открой мне файл .env». Он сам всё откроет и подставит.

5
Проверь работу

Открой своего бота в Telegram, нажми Start и напиши сообщение.

Если что-то пошло не так

Задавай вопросы Claude и присылай скриншоты своих ошибок с комментарием «Помоги» — он разберётся.
