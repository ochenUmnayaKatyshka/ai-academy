# AI Academy — Спецификация сайта

---

## 1. Общее

- **Название:** AI Academy
- **Подзаголовок:** От нуля до создания AI-продуктов
- **Тип:** Обучающий портал (статический сайт)
- **Язык контента:** Русский
- **Тема:** Тёмная (чёрный фон, серые карточки)
- **Стек:** Astro + Tailwind CSS
- **Контент:** берётся из существующих .md файлов курса

Открывается при клике на карточку урока. Структура сверху вниз:

```
┌────────────────────────────────────────────┐
│  ← Назад               Этап 1 · Урок 3/8  │
├────────────────────────────────────────────┤

│  # Файлы и форматы                         │
│                                            │
│  > Вводная цитата урока (из .md файла)     │

│  ┌────────────────────────────────────┐    │
│  │                                    │    │
│  │          [ ▶ Play ]                │    │
│  │                                    │    │
│  │    встроенный видеоплеер           │    │
│  │    (YouTube / загруженный файл)    │    │

│  - Заголовки, текст                        │

│  - Таблицы                                 │

│  ○ Программа для видео                     │
│  ○ Текстовый интерфейс для управления      │

│                                            │
│  Вопрос 2: Команда pwd показывает...       │

│                                            │
│  ← Предыдущий урок    Следующий урок →     │
│                                            │
└────────────────────────────────────────────┘
```

### Порядок блоков на странице урока:

1. **Шапка** — кнопка "← Назад", метка "Этап X · Урок Y из Z"
2. **Название урока** — заголовок из .md файла
3. **Вводная** — цитата (блок `>` из .md файла)
4. **Видео** — встроенный видеоплеер (YouTube embed или загруженный файл)
5. **Материал** — полный контент из .md файла
6. **Тест** — 3-5 вопросов с вариантами ответов. Только после прохождения теста урок отмечается пройденным
7. **Навигация** — кнопки "← Предыдущий" / "Следующий →"

### Тест — подробности

- 3-5 вопросов с вариантами ответов (radio buttons)
- Вопросы берутся из контента урока
- Кнопка "Проверить ответы"
- Показывает результат: "4/5 правильно"
- Если ≥ 80% правильно → урок автоматически отмечается пройденным ✅
- Если < 80% → можно пересдать ("Попробовать ещё раз")
- Пока тест не пройден — урок НЕ считается пройденным

### Видео — подробности

- Встроенный YouTube плеер прямо на странице (iframe embed)
- Видео проигрывается на сайте, не перенаправляет на YouTube
- Плеер на всю ширину контента
- Видео — необязательный блок (если нет видео для урока — блок не показывается)
- Видео добавляется позже, после запуска сайта

## 7. Цвета и стиль

| Элемент | Цвет |
|---------|------|
| Фон страницы | `#0a0a0a` |
| Фон карточки | `#141414` |
| Рамка карточки | `#2a2a2a` |
| Рамка hover | `#6b8afd` |
| Текст основной | `#e5e5e5` |
| Текст вторичный | `#888888` |
| Заголовки секций | `#666666` uppercase |
| Акцент | `#6b8afd` (голубой) |
| Пройдено | `#4ade80` (зелёный) |
| Ошибка (тест) | `#f87171` (красный) |
| Блоки кода | `#1a1a1a` |

- **Шрифт:** Inter (или системный sans-serif), код — JetBrains Mono
- **Скругления:** 12px карточки, 8px кнопки, 6px код
- **Тени:** нет
- **Анимации:** мягкий hover transition 0.2s
- **Ширина контента:** max 640px на главной, max 720px на странице урока

## 8. Поведение

| Что | Как |
|-----|-----|
| Прогресс | localStorage (без регистрации) |
| Замочки | Нет — всё открыто сразу |
| Отметить пройденным | Только через тест (≥80% правильных) |
| Подсветка кода | Да (Python, bash, json) |
| Копировать код | Да, кнопка в углу блока |
| Спойлеры | Аккордеон, закрыт по умолчанию |
| Мобилка | Адаптивная, карточки 2 колонки |
| Видео | Опционально, если нет — блок скрыт |
