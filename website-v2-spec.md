# AI Academy — Website V2 Spec

## Стиль
Вдохновение: Gonka.ai — брутальный минимализм, editorial, чистый и дорогой.

---

## 1. Общий вайб
- Светлый фон (не тёмный!) — off-white `#f5f5f0`
- **Фоновый паттерн:** полупрозрачная точечная сетка (dot grid) поверх фона — мелкие точки с равным шагом, opacity ~5-8%, цвет `#000`. Создаёт текстуру "бумаги" / "технической сетки"
- Минимум цветов: чёрный текст, светлый фон, один яркий акцент
- Много воздуха (whitespace)
- Чёткие линии и бордеры, без размытий и теней
- Плоский дизайн — без градиентов, glow, стекла

## 2. Шрифты
- **Заголовки:** жирный, крупный, uppercase. Шрифт: `Space Grotesk` или `Inter` (800 weight)
- **Текст и описания:** моноширинный шрифт `JetBrains Mono` или `Space Mono`
- Заголовки — чёрные `#0a0a0a`, описания — серые `#666`

## 3. Акцентный цвет
- Лаймовый/кислотно-жёлтый `#c8e600` — как на референсе
- Используется для: бейджей, выделения активной карточки, номеров этапов, прогресс-бара
- Альтернатива (обсудить): оставить лайм или заменить на другой?

## 4. Структура главной страницы

### 4.1 Верхняя полоса (announcement bar)
- Тонкая полоска сверху с текстом, например: "30 УРОКОВ · ОТ НУЛЯ ДО AI-ПРОДУКТОВ · БЕСПЛАТНО"
- Чёрный фон, белый текст, моноширинный шрифт, мелкий размер

### 4.2 Навигация
- Логотип слева (текст "AI ACADEMY" или иконка)
- Переключатель темы справа (☀/🌙)
- Чистая, минимальная

### 4.3 Hero (главный блок)
- Бейдж сверху: `[🎓 AI ACADEMY — КУРС ДЛЯ ТЕХ, КТО НИКОГДА НЕ КОДИЛ]` — чёрный фон, белый текст, маленький прямоугольник
- Огромный заголовок (uppercase, bold):
  ```
  НАУЧИСЬ СОЗДАВАТЬ
  AI-ПРОДУКТЫ \\
  ```
  (с декоративными слэшами \\ как на референсе)
- Подзаголовок: моноширинный, серый
  "Пошаговый путь от нуля. 30 уроков, 5 этапов, практика на каждом шаге."

### 4.4 Прогресс
- Полоса прогресса — тонкая, с лаймовым заполнением
- Текст: "0 из 30 уроков" — моноширинный

### 4.5 Большие карточки (Обучение / Материалы)
- 2 карточки в ряд
- Чёрный бордер 1-2px, без фона (или чуть светлее)
- **При hover:** карточка увеличивается (scale ~1.03-1.05), бордер становится лаймовым `#c8e600`, лёгкая лаймовая тень (box-shadow). Плавная анимация ~0.2s
- **При клике (active):** фон становится лаймовым `#c8e600`, текст чёрный, scale чуть меньше (1.02) — эффект нажатия
- Заголовок — uppercase, bold

### 4.6 Этапы с уроками
- Каждый этап:
  - Номер этапа в лаймовом бейдже: `01` `02` `03` `04` `05`
  - Заголовок — uppercase, крупный, bold
  - Описание — моноширинный, серый
  - Счётчик — справа, моноширинный `0/8`
- **Карточки уроков — layout "карточка + описание":**
  - Каждый урок — горизонтальный ряд на всю ширину:
    - **Слева:** карточка-кнопка (эмодзи + название урока). Чёрный бордер, hover — лаймовый. Кликабельная, ведёт на урок.
    - **Справа:** текст "Что узнаешь:" — короткое описание (1-2 строки), что человек будет знать/уметь после урока. Моноширинный шрифт, серый цвет.
  - Уроки идут вертикальным списком (не сетка 2 колонки)
  - Completed: лаймовый бордер на карточке + галочка

### 4.6.1 Описания уроков (для правой части карточки)

**Этап 1 — Фундамент**
| Урок | Описание |
|------|----------|
| Подготовка | Установишь всё, что нужно для работы |
| Терминал | Сможешь управлять компьютером без мышки |
| Файлы и форматы | Поймёшь, чем .py отличается от .json |
| Переменные и типы | Напишешь первый код на Python |
| Условия и циклы | Научишь программу принимать решения |
| Функции | Перестанешь копировать один и тот же код |
| Работа с файлами | Сможешь читать и записывать данные |
| Git и GitHub | Сохранишь код так, чтобы ничего не потерять |

**Этап 2 — Понимание AI**
| Урок | Описание |
|------|----------|
| Что такое LLM | Поймёшь, как ChatGPT генерирует текст |
| API — язык программ | Узнаешь, как программы общаются друг с другом |
| Первый вызов Claude API | Отправишь запрос к AI из своего кода |
| Prompt Engineering | Научишься получать от AI точные ответы |
| Токены и стоимость | Поймёшь, сколько стоит каждый запрос |
| Скрипты с Claude API | Автоматизируешь рутину с помощью AI |

**Этап 3 — AI-агенты**
| Урок | Описание |
|------|----------|
| Чат-бот vs Агент | Поймёшь разницу между болтуном и исполнителем |
| Tools — руки агента | Дашь AI возможность действовать в реальном мире |
| Память агента | Научишь AI помнить контекст разговора |
| Первый агент (SDK) | Соберёшь работающего агента с нуля |
| Мульти-шаговые задачи | Агент будет сам планировать и выполнять цепочки |

**Этап 4 — Продукты**
| Урок | Описание |
|------|----------|
| Telegram-бот с AI | Запустишь бота, с которым можно общаться |
| Базы данных | Научишься хранить данные пользователей |
| Сервер и деплой | Выложишь проект в интернет |
| Платформы и OpenClaw | Узнаешь, какие инструменты ускоряют разработку |
| Безопасность | Защитишь свой продукт от основных угроз |

**Этап 5 — Эксперт**
| Урок | Описание |
|------|----------|
| RAG — поиск по данным | AI будет отвечать на основе твоих документов |
| Мульти-агентные системы | Несколько AI будут работать вместе |
| Fine-tuning | Настроишь модель под свою задачу |
| Оптимизация и стоимость | Сократишь расходы на AI в разы |
| Архитектура AI-продуктов | Спроектируешь систему как профессионал |

### 4.7 Footer
- Тонкая линия сверху
- Текст: "В создании обучения участвовали:"
- Фото, имя, роль, Instagram
- Копирайт — мелкий, моноширинный

## 5. Страница урока
- Та же стилистика: светлый фон, моно-шрифт
- Хедер: "← НАЗАД К КУРСУ" (uppercase) + "ЭТАП 1 / УРОК 3"
- Контент из markdown — стилизованный под общий стиль
- Код: тёмный блок с моноширинным шрифтом, лаймовая кнопка "КОПИРОВАТЬ"
- Навигация внизу: "← ПРЕДЫДУЩИЙ" / "СЛЕДУЮЩИЙ →"

## 6. Переключатель тем

### Dark (нормальный пацан)
- Фон: `#0a0a0a`
- Текст: `#e5e5e5`
- Карточки: бордер `#333`, фон `#111`
- Акцент: лайм `#c8e600` (тот же)
- Шрифты: те же

### Girls (для девочек)
- Фон: `#f5f0ed` (тёплый кремовый)
- Текст: `#3a3a3a`
- Шрифт: `Georgia`, serif
- Акцент: розовый `#c47a8a`
- Карточки: белые, тонкий бордер, hover розовый
- Стиль: элегантный, мягкий

### Light (по умолчанию)
- Фон: `#f5f5f0` (холодный off-white)
- Стиль из референса Gonka — это и есть основная тема

## 7. Поведение
- Прогресс: localStorage, как раньше
- Анимации: минимальные, не flashy. Только hover-переходы на карточках.
- Тема: сохраняется в localStorage
- Все ссылки рабочие (localhost сервер)

- Материалы (контент шпаргалок/глоссария)
