# Prompt Engineering — Шпаргалка для печати

---

## Формула идеального промпта

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  [РОЛЬ]          Ты — {кто}.                        │
│  [КОНТЕКСТ]      Вот данные / ситуация: {что}.      │
│  [ЗАДАЧА]        Сделай {что именно}.               │
│  [ФОРМАТ]        Ответь в формате {каком}.          │
│  [ОГРАНИЧЕНИЯ]   НЕ делай {чего}. Длина: {X}.      │
│  [ПРИМЕРЫ]       Вот пример: {пример}.              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## 9 техник промптинга

| # | Техника | Что делает | Когда использовать | Пример |
|---|---------|------------|-------------------|--------|
| 1 | **Роль** | Задаёт "характер" AI | Всегда | `system: "Ты — бизнес-аналитик"` |
| 2 | **Few-shot** | Показываешь 2-3 примера ответа | Когда нужен точный формат | `Вход: "текст" → Выход: {json}` |
| 3 | **Chain of thought** | Просишь думать пошагово | Сложные задачи, расчёты | `"Рассуждай пошагово в <thinking>"` |
| 4 | **XML-теги** | Структурируешь промпт | Сложные промпты с данными | `<instructions>`, `<document>` |
| 5 | **Разделители** | Отделяешь данные от инструкции | Когда даёшь текст для обработки | `---` или `"""` между блоками |
| 6 | **Справочный текст** | Даёшь источник, снижаешь выдумки | Нужны точные факты | `"Отвечай ТОЛЬКО по тексту"` |
| 7 | **Самопроверка** | Просишь перепроверить ответ | Критичные задачи | `"Перепроверь свой ответ"` |
| 8 | **Prefill** | Начинаешь ответ за AI | Строгий формат (JSON) | Начать ответ: `{"name": "` |
| 9 | **Подзадачи** | Разбиваешь большое на маленькое | Большие задачи | 3 запроса вместо 1 |

## Плохой промпт → Хороший промпт

| Плохой | Хороший |
|--------|---------|
| Напиши пост | Напиши пост для Instagram про AI, 3 абзаца, для предпринимателей, с вопросом в конце |
| Проанализируй данные | Найди топ-3 товара по выручке, тренд роста/падения, ответь таблицей |
| Напиши код | Напиши Python-функцию, которая фильтрует email-адреса, с комментариями |
| Помоги с бизнесом | Я запускаю AI-бота для крипто-чатов. Подписка или разовая оплата? Аудитория — майнеры |
| Переведи | Переведи текст ниже на английский. Дай только перевод, без пояснений |

## Частые ошибки

| Ошибка | Исправление |
|--------|-------------|
| Слишком размыто | Добавь контекст, формат, аудиторию |
| Нет примеров | Покажи 2-3 примера ответа |
| Всё в одном промпте | Разбей на цепочку запросов |
| AI выдумывает факты | Дай справочный текст + "отвечай ТОЛЬКО по нему" |
| Нет ограничений | Скажи чего НЕ делать |
| Формат не указан | Опиши формат: JSON, таблица, список |
| Данные и инструкции смешаны | Используй XML-теги или разделители |

## Шаблоны для копирования

### Анализ текста

```
<document>
[вставь текст]
</document>

<instructions>
На основе документа выше:
1. [вопрос 1]
2. [вопрос 2]
3. [вопрос 3]

Если информации нет в тексте — напиши "не указано".
Формат ответа: таблица.
</instructions>
```

### Извлечение данных (лиды, контакты)

```
Извлеки данные из сообщения.

Примеры:
Сообщение: "Иван, @ivan_s, ищу поставщика ASIC"
Результат: {"name": "Иван", "contact": "@ivan_s", "request": "поставщик ASIC"}

Сообщение: "Привет всем, как дела?"
Результат: ПРОПУСТИТЬ

Сообщение: "[вставь сообщение]"
Результат:
```

### Генерация контента

```
Ты — [роль].

Аудитория: [кто будет читать]
Тон: [какой стиль]
Длина: [сколько]

Задача: [что написать]

НЕ используй: [что исключить]
Обязательно включи: [что должно быть]
```

### Пошаговый анализ

```
[вопрос или задача]

Рассуждай пошагово:
<thinking>
Шаг 1: ...
Шаг 2: ...
Шаг 3: ...
</thinking>

<answer>
Финальный ответ.
</answer>
```

## Быстрая проверка промпта

Перед отправкой проверь:

- [ ] Есть роль? (кто отвечает)
- [ ] Есть контекст? (что AI должен знать)
- [ ] Задача конкретная? (не "помоги", а "найди 3 проблемы")
- [ ] Указан формат? (таблица, JSON, список)
- [ ] Есть ограничения? (чего НЕ делать)
- [ ] Есть примеры? (хотя бы 1-2)
- [ ] Данные отделены от инструкции? (теги или разделители)
