# Калькулятор стоимости AI — Шпаргалка для печати

---

## Зачем считать стоимость AI

Когда ты создаёшь AI-продукт — бота, ассистента, систему аналитики — тебе нужно знать, сколько он будет стоить **до того**, как ты его запустишь. Иначе можно получить счёт на $500 в конце месяца и не понять, откуда он взялся.

### Что это даёт:

| Зачем считать | Что это значит для бизнеса |
|---------------|---------------------------|
| **Планирование бюджета** | Ты знаешь заранее, сколько потратишь в месяц — нет сюрпризов |
| **Ценообразование** | Если продаёшь продукт — знаешь свою себестоимость и можешь поставить правильную цену |
| **Выбор модели** | Понимаешь, где Haiku хватит, а где нужен Sonnet — и не переплачиваешь |
| **Оптимизация** | Видишь, что именно стоит дорого — и можешь это исправить |
| **Масштабирование** | Знаешь, сколько будет стоить при 100 пользователях, при 1000, при 10000 |

**Пример:** Ты запускаешь бота за $2/мес и берёшь с клиентов $10/мес подписку. 100 клиентов = $1000 дохода - $200 расходов = **$800 прибыли**. Но если не посчитать заранее и выбрать Opus вместо Haiku, расходы будут $2000 — и ты уйдёшь в минус.

## Формула расчёта

```
Стоимость 1 запроса =
  (входные токены × цена input / 1,000,000)
  + (выходные токены × цена output / 1,000,000)

Стоимость в день = стоимость 1 запроса × количество запросов в день

Стоимость в месяц = стоимость в день × 30
```

## Готовые расчёты типичных проектов

### Бот-монитор чата (твой OpenClaw)

```
Модель: Haiku
Задача: классификация сообщений, извлечение лидов
Input: ~400 токенов (system prompt + сообщение)
Output: ~100 токенов (JSON или "НЕ ЛИД")
Запросов: 200/день

Input:  200 × 400 = 80,000 токенов × $0.80 / 1M = $0.064
Output: 200 × 100 = 20,000 токенов × $4.00 / 1M = $0.080
В день: $0.14
В месяц: $4.20
```

### Telegram-бот для поддержки клиентов

```
Модель: Sonnet
Задача: отвечать на вопросы клиентов
Input: ~800 токенов (system prompt + история + вопрос)
Output: ~400 токенов (развёрнутый ответ)
Запросов: 100/день

Input:  100 × 800 = 80,000 токенов × $3.00 / 1M = $0.24
Output: 100 × 400 = 40,000 токенов × $15.00 / 1M = $0.60
В день: $0.84
В месяц: $25.20
```

### Генератор Instagram-постов

```
Модель: Sonnet
Задача: генерация постов по теме
Input: ~500 токенов (system prompt + тема)
Output: ~800 токенов (пост + хештеги)
Запросов: 5/день

Input:  5 × 500 = 2,500 токенов × $3.00 / 1M = $0.008
Output: 5 × 800 = 4,000 токенов × $15.00 / 1M = $0.060
В день: $0.07
В месяц: $2.10
```

### AI-чат с памятью (длинные разговоры)

```
Модель: Sonnet
Задача: персональный ассистент
Input: ~3,000 токенов (растёт с историей!)
Output: ~500 токенов
Запросов: 30/день

Input:  30 × 3,000 = 90,000 токенов × $3.00 / 1M = $0.27
Output: 30 × 500 = 15,000 токенов × $15.00 / 1M = $0.225
В день: $0.50
В месяц: $15.00
```

## API vs Подписка: когда что выгоднее

| Ситуация | API стоимость | Подписка | Что выгоднее |
|----------|---------------|----------|--------------|
| Бот-монитор (200 запросов/день, Haiku) | ~$4/мес | $20/мес | **API в 5 раз дешевле** |
| Генератор постов (5/день, Sonnet) | ~$2/мес | $20/мес | **API в 10 раз дешевле** |
| Личный ассистент (30/день, Sonnet) | ~$15/мес | $20/мес | **Примерно одинаково** |
| Активное личное использование (80+/день) | ~$30+/мес | $20/мес | **Подписка дешевле** |
| Публичный бот (1000 запросов/день, Sonnet) | ~$180/мес | Невозможно | **Только API** |

## 5 способов снизить расходы

| # | Способ | Экономия | Пример |
|---|--------|----------|--------|
| 1 | **Начать с Haiku** вместо Sonnet | до 80% | Классификация, FAQ, фильтрация |
| 2 | **Ограничить историю чата** | до 50% | `messages=history[-10:]` вместо всей истории |
| 3 | **Prompt caching** | до 90% на input | Одинаковый system prompt в каждом запросе |
| 4 | **Batch API** | 50% на всё | Обработка данных ночью, без срочности |
| 5 | **Писать промпты на английском** | 30-40% на input | System prompt на английском |
