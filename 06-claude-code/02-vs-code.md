# Установка VS Code — твоя рабочая среда

Прежде чем запускать Claude Code, нужно всё удобненько настроить. Чтобы можно было делать продукты в одном окне — никуда не выходить и не отвлекаться.

---

## Что такое VS Code и зачем он нужен

**VS Code** (полное название — Visual Studio Code) — это бесплатная программа для работы с кодом. Но не пугайся слова «код» — по сути это текстовый редактор, который пишет твой продукт.

VS Code:

- **Бесплатный** — полностью, без скрытых платежей

- **Работает на Mac, Windows и Linux** — одинаково на всех системах

- **Лёгкий** — запускается быстро, не тормозит

- **Расширяемый** — можно ставить дополнения (расширения), которые добавляют новые функции

### Почему именно VS Code, а не другие редакторы?

Существуют десятки редакторов кода: Sublime Text, Atom, WebStorm, Cursor и другие. Они работают плюс-минус одинаково. Когда умеешь работать хотя бы с одним из редакторов (главное суть понять) — всегда можно переключиться на любой другой.

Мы используем VS Code по трём причинам:

1. У него **самое большое сообщество** — если возникнет проблема, ответ найдётся за минуту

2. Claude Code имеет **официальное расширение** именно для VS Code (чатик будет работать прямо внутри)

3. Он бесплатный и его **не нужно настраивать** — работает из коробки (за Cursor, например, нужно платить отдельно от оплаты моделей — ещё +$20)

## Установка на Mac

**ШАГ 1: Скачать**

1. Открой браузер и перейди на **code.visualstudio.com**

![Сайт VS Code — кнопка Download](../img/cc/01-vscode-download.png)

2. Нажми большую кнопку **Download for macOS**

3. Скачается файл типа `VSCode-darwin-arm64.zip` (или похожий)

**ШАГ 2: Установить**

1. Открой папку **Загрузки** (Downloads)

2. Найди скачанный `.zip` файл и дважды кликни — он распакуется

3. Появится файл **Visual Studio Code** с синей иконкой

4. Перетащи его в папку **Программы** (Applications)

> 💡 **Зачем в Программы:**
> Можно запускать и из Загрузок, но тогда при каждом обновлении будут проблемы. В Программах всё работает как надо.

**ШАГ 3: Запустить**

1. Открой **Finder** → **Программы** (Applications)

2. Найди **Visual Studio Code** и дважды кликни

3. Может появиться предупреждение «Программа загружена из интернета» — нажми **Открыть**

**Проверка:**

- ✅ Открылось окно VS Code с тёмным интерфейсом

- ✅ Слева панель с иконками

- ✅ В центре приветственная вкладка

## Установка на Windows

**ШАГ 1: Скачать**

1. Открой браузер и перейди на **code.visualstudio.com**

2. Нажми большую синюю кнопку **Download for Windows**

3. Скачается файл типа `VSCodeUserSetup-x64.exe`

**ШАГ 2: Установить**

1. Открой папку **Загрузки**

2. Запусти скачанный `.exe` файл

3. Появится установщик — жми **Далее** на каждом шаге

4. На шаге «Выберите дополнительные задачи» поставь галочку **Добавить в PATH**

5. Нажми **Установить** и дождись завершения

> ⚠️ **Важно:**
> Галочка «Добавить в PATH» — обязательна. Без неё Claude Code не сможет нормально работать. Если пропустил — переустанови.

**ШАГ 3: Запустить**

1. Нажми **Win** (кнопка с флажком Windows)

2. Напиши **VS Code** или **Visual Studio Code**

3. Кликни на появившуюся программу

**Проверка:**

- ✅ Открылось окно VS Code с тёмным интерфейсом

- ✅ Слева панель с иконками

- ✅ В центре приветственная вкладка
