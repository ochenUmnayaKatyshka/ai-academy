# Твой первый проект

Прежде чем что-то делать с Claude — нужно место где это делать. Проект. Папка на компьютере, которую откроешь в VS Code.

---

## Где хранить

Создай место для будущих проектов:

**Рекомендую:**
- `Документы/Projects/` — папка для всех проектов
- Внутри — отдельная папка для каждого проекта

> ⚠️ **Очень важно:** называй все папки на английском языке. Просто запомни как правило — русские названия папок вызывают ошибки в куче инструментов. Английский тут не про язык, а про совместимость.

Пример структуры:

```
Документы/
  Projects/
    my-first-project/    ← твой первый проект
    telegram-bot/
    landing-page/
```

Так всё будет в одном месте и не потеряется.

> 💡 **Совет:**
> Создай папку Projects прямо сейчас. Всё остальное будем делать внутри неё.

---

## Что такое проект

Это просто папка с файлами. Всё что Claude создаст — код, стили, документы — сохраняется туда.

Почему это важно:
- **Порядок** — все файлы проекта в одном месте
- **Контекст** — Claude видит все файлы в папке и понимает связи
- **История** — можно вернуться к старым версиям (об этом позже)

Один проект = одна папка = одна идея.

---

## Создание папки

**На Mac:**
1. Открой Finder
2. Перейди в Документы
3. Нажми `Cmd + Shift + N` (или правой кнопкой → «Новая папка»)
4. Назови: `my-first-project`

**На Windows:**
1. Открой Проводник
2. Перейди в Документы
3. Нажми `Ctrl + Shift + N` (или правой кнопкой → «Создать папку»)
4. Назови: `my-first-project`

> 💡 **Совет по названию:**
> Без пробелов и русских букв. Используй дефис или подчёркивание:
> ✅ `my-project`
> ✅ `task_manager`
> ❌ `мой проект`
> ❌ `My Project!!!`

---

## Открытие в VS Code

**Способ 1 — через VS Code:**
1. Открой VS Code
2. **File → Open Folder** (или `Cmd/Ctrl + O`)
3. Найди свою папку и выбери её
4. Нажми **Open**

**Способ 2 — перетаскиванием:**
- Перетащи папку прямо на окно VS Code

**Способ 3 — из Finder/Проводника:**
- Правой кнопкой на папке → «Открыть с помощью» → VS Code

---

## Что ты видишь

После открытия:
- **Слева** — панель файлов. Пока пусто — это нормально. Здесь будут появляться файлы от Claude
- **Справа** — пустая область для открытых файлов
- **Внизу** — строка состояния

> ⚠️ **Важно:**
> Убедись что папка открыта как проект. Сверху в заголовке окна должно быть имя папки. Если там «Welcome» или пусто — повтори шаги.

---

## Проверка Claude Code

1. Открой панель Claude Code: `Cmd/Ctrl + Shift + P` → «Claude Code: Open in sidebar»
2. Справа откроется чат с полем для ввода
3. Напиши: **Привет! Ты меня слышишь?**
4. Нажми Enter

Claude ответит — значит всё работает.

> 💡 **Важно:**
> Claude ответит даже без открытого проекта. Но для работы с файлами нужна открытая папка. Проверь что слева видно имя папки.

---

- ✅ Открыта в VS Code
- ✅ Вижу панель файлов слева (пока пустую)
- ✅ Claude отвечает на сообщения
